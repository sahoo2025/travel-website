# Wanderlust Travels - Professional Travel Website

A modern, responsive travel website built with HTML, CSS, and JavaScript. Features destination listings, travel packages, contact forms, and more.

## 🌟 Features

- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **5 Complete Pages**: Home, About Us, Destinations, Packages, Contact
- **Interactive Elements**: Search/filter functionality, form validation, testimonial slider
- **Modern UI**: Clean design with smooth animations and hover effects
- **SEO Optimized**: Semantic HTML5 structure
- **Fast Performance**: Optimized CSS and vanilla JavaScript

## 🌐 Deployment

This website is designed for GitHub Pages deployment. Follow these steps:

### Option 1: Automatic Deployment (Recommended)
1. Push your code to GitHub
2. Go to repository Settings > Pages
3. Select "Deploy from a branch"
4. Choose `main` branch and `/(root)` folder
5. Click Save

### Option 2: Manual Deployment
1. Build your project (if needed)
2. Push to GitHub
3. Enable GitHub Pages in repository settings

## 🔧 Customization

### Changing Colors
Edit the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #007bff;
    --secondary-color: #0056b3;
    --text-color: #333;
    --background-color: #fff;
}
```

### Adding New Destinations
1. Add new destination cards to `destinations.html`
2. Update the filter categories if needed
3. Add corresponding images

### Updating Packages
1. Modify package information in `packages.html`
2. Update pricing and features
3. Add new package categories as needed
